## The useful part of cyber style

Cyber visuals work best when they make information feel live. The glow, grid, and motion should support scanning instead of competing with the content.

## Rules for this version

- Keep text contrast high.
- Use animation as atmosphere, not as the main event.
- Make cards compact enough for repeated updates.
- Let the content model stay simple.

The next design pass can add richer project media, real screenshots, and better visual links between software, writing, and research.
